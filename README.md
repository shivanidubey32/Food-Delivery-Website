# Food Delivery Website 🍔

A responsive Food Delivery Web Application built using React and Tailwind CSS.  
Users can browse food items, add them to the cart, and manage their orders easily.

## 🌐 Live Demo
https://shi-food-delivery.netlify.app

## 🚀 Features
- Browse food items
- Add and remove items from cart
- Dynamic cart management
- Responsive UI

## 🛠 Tech Stack
- React.js
- Tailwind CSS
- JavaScript
- Vite
